A WordPress plugin for Amazing System.  I'll flesh this out with more
information if someone asks nicely. (and actually reads readme files)

```php
AmSys_Options::get( 'key.subkey' ); // User options
AmSys_Config::get( 'key.subkey' ); // Developer configuration
AmSys_State::get( 'key.subkey' ); // Current state of the plugin
AmSys_Info::get( 'Name' ); // Plugin
```
